Projeto Correndo pelo Futuro

Esta página apresenta detalhes sobre o nosso projeto e como você pode participar ou contribuir.

Sobre o Projeto

O Projeto Correndo pelo Futuro é voltado para crianças e jovens, incentivando a prática de corrida de rua como forma de promover saúde, disciplina e integração comunitária.

Como Participar

  • Inscreva-se como participante em nossas corridas comunitárias.
  • Torne-se voluntário para ajudar na organização dos eventos.
  • Contribua com doações para apoiar nossos projetos.
  • Divulgue o projeto para familiares e amigos.
